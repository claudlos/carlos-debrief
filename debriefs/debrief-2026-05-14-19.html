📰 News Headlines

  • M5 Kernel Exploit: Security researchers built the first public macOS kernel memory corruption exploit against Apple's M5 chip in just five days, despite Apple's five-year hardware hardening effort — a stark reminder that hardware security promises remain work-in-progress.
  • LLM Inference Efficiency: LEAD (Length-Efficient Adaptive Dynamic reasoning) tackles the verbosity problem in large reasoning models like OpenAI o1 and DeepSeek-R1, using online self-adaptive mechanisms to achieve better accuracy at lower compute cost.
  • AI Agent Evaluation Gap: A study finds frontier coding assistants achieve only 30% execution success when automating agent evaluation without domain expertise, prompting EvalAgent — a new pipeline raising first-run success to 65%.
  • Citation Hallucination Detection: CiteTracer, a multi-agent detector with a 12-code taxonomy, reaches 97.1% accuracy on synthetic benchmarks and detects 97.1% of real-world fabrications from ICLR 2026, releasing a 2,450-citation benchmark.
  • Digital Sovereignty Shift: A Lobste.rs thread on moving digital infrastructure to European cloud highlights growing practitioner interest in data sovereignty beyond US hyperscalers.
