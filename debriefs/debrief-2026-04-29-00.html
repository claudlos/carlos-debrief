⚡ Quick Summary

  • Fine-tuning LLMs on new factual data amplifies hallucinations through semantic interference between overlapping representations; self-distillation regularization can mitigate this without sacrificing new knowledge acquisition.
  • Personality-conditioned LLM personas (especially Dark Triad traits) significantly amplify gender bias in generated narratives across English and Hindi, with harm varying unpredictably by model and language.
  • Credal Concept Bottleneck Models (CREDENCE) decompose concept-level uncertainty into epistemic (reducible) and aleatoric (irreducible) components, enabling systems to route uncertain cases to human review.
  • Table retrieval systems are highly sensitive to serialization format (CSV vs HTML vs Markdown), with centroid-based representation averaging suppressing format-specific noise and improving cross-format robustness.
  • A CVSS 8.7 RCE vulnerability (CVE-2026-3854) in GitHub Enterprise Server allows unauthenticated remote code execution via a critical attack vector in GHES-3.x.
