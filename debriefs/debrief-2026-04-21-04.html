⚡ Quick Summary

  • MathNet Benchmark: A massive multimodal multilingual dataset of Olympiad-level math problems spanning 47 countries and 17 languages, providing the first rigorous test of mathematical reasoning across both generative models and retrieval systems.
  • Sessa (Selective SSM Attention): A new state-space model architecture that addresses the diffuse-attention problem in standard SSMs, where old token influence scales as O(1/N), by adding selective retrieval mechanisms — potentially a more efficient alternative to Transformers for long contexts.
  • BLF (Bayesian Linguistic Forecaster): An agentic binary forecasting system combining Bayesian probability estimates with LLM-generated natural-language evidence summaries, achieving state-of-the-art on ForecastBench through iterative tool-use and belief updating.
  • LPSR (Latent Phase-Shift Rollback): An inference-time error correction method that monitors the residual stream for abrupt directional reversals (phase shifts) and rolls back the KV-cache to correct unrecoverable mid-generation reasoning errors before they compound.
  • Privacy-Preserving ANN Search: A framework combining Fully Homomorphic Encryption (FHE) with a Trusted Execution Environment (TEE) to enable approximate nearest-neighbor search on confidential datasets without exposing embeddings — solving a key bottleneck for privacy-preserving LLM deployments.

📰 News Headlines

  • Formal Verification of Signal Protocol: A major new project (Signal Shot) aims to verify the Signal encryption protocol and its Rust implementation using the Lean proof assistant — a significant step toward cryptographic implementation correctness.
  • Terminal Drag-and-Drop Command Injection: Security researchers disclosed that dragging files into terminal emulators like Kitty and xfce4-terminal inserts the file path at the cursor, which can be exploited to execute arbitrary commands — a relevant attack surface as AI coding assistants become more integrated with terminal workflows.
  • ChatGPT + Cloudflare React State Surveillance: A technical analysis reveals that ChatGPT waits for Cloudflare to read the user's React component state before allowing input — raising new privacy questions about third-party telemetry embedded in AI interfaces.
  • Vulnerability Sightings Forecasting: A new study on forecasting sparse, bursty vulnerability-related activity (CVE discussions, PoC releases) finds that count-based methods like Poisson regression outperform traditional time-series approaches for short-horizon predictions.
  • LoRA Merging Breakthrough: Research identifying that LoRA merge interference stems from the output matrix B repeatedly using a small set of shared directions across tasks — introducing Pico, a pre-merge calibration method that improves merged adapter accuracy by 3.4–8.3 points.
