## ArXiv Papers

### 🧠 LLMs

**1. Sublinear Memory for Large-Scale Genomic Data**
- **Authors**: Chuangtao Chen, Grace Li Zhang, Xunzhao Yin, Cheng Zhuo, Bing Li
- **Description**: Proposes KV Packet, a recomputation-free cache reuse framework that treats cached documents as immutable packets wrapped in lightweight trainable soft-token adapters, bridging context discontinuities through self-supervised distillation.
- **Why it matters**: Achieves near-zero FLOPs and lower TTFT than recomputation-based baselines while retaining F1 scores comparable to full recomputation, addressing a major bottleneck in LLM inference efficiency.
- **Link**: https://huggingface.co/papers/2604.13226

**2. Memory Transfer Learning for Coding Agents**
- **Authors**: Kangsan Kim, Minki Kang, Taeil Kim, Yanlai Yang, Mengye Ren
- **Description**: Investigates Memory Transfer Learning (MTL) by harnessing a unified memory pool from heterogeneous domains, evaluating performance across 6 coding benchmarks using four memory representations.
- **Why it matters**: Demonstrates that cross-domain memory transfer improves average performance by 3.7%, primarily through transferring meta-knowledge like validation routines rather than task-specific code.
- **Link**: https://huggingface.co/papers/2604.14004

### 🤖 Machine Learning

**3. ROSE: Intent-Centered Evaluation Metric for NL2SQL**
- **Authors**: Wenqi Pei, Shizheng Hou, Boyan Li, Han Chen, Zhichao Shi
- **Description**: Introduces ROSE, an intent-centered metric that focuses on whether predicted SQL answers the question using an adversarial Prover-Refuter cascade, addressing limitations of execution accuracy metrics.
- **Why it matters**: Achieves best agreement with human experts, outperforming next-best metric by nearly 24% in Cohen's Kappa, providing more reliable evaluation for NL2SQL systems.
- **Link**: https://huggingface.co/papers/2604.12988

**4. Temporal Graph Attention for Affordance Classification**
- **Authors**: Ami Chopra, Supriya Bordoloi, Shyamanta M. Hazarika
- **Description**: Presents EEG-tGAT, a temporally augmented Graph Attention Network that incorporates temporal attention to modulate contribution of different time segments and temporal dropout for regularization.
- **Why it matters**: Explicitly encodes temporal importance and enforces temporal robustness, showing that modest architectural changes to graph attention models yield consistent benefits for affordance-driven interaction data.
- **Link**: https://huggingface.co/papers/2604.10149

### 🛡️ AI Safety

**8. Activation-Scaling Guard (ASGuard) for Targeted Jailbreaking Attack Mitigation**
- **Authors**: Yein Park, Jungwoo Park, Jaewoo Kang
- **Description**: Introduces ASGuard, a mechanismically-informed framework using circuit analysis to identify attention heads linked to targeted jailbreaking, then trains precise channel-wise scaling vectors to recalibrate vulnerable head activations.
- **Why it matters**: Effectively reduces attack success rate while preserving general capabilities and minimizing over-refusal, achieving Pareto-optimal balance between safety and utility in LLM alignment.
- **Link**: https://huggingface.co/papers/2509.25843
