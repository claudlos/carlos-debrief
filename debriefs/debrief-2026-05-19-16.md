## 🌐 Web Findings

### 🤗 HuggingFace Papers

**AR-VLA: True Autoregressive Action Expert for Vision-Language-Action Models**  
Proposes a standalone autoregressive Action Expert for robotic control that maintains its own long-lived memory history, unlike reactive VLA models that reset with each observation. Uses a re-anchoring mechanism to handle perception staleness and achieves smoother, more spatio-temporally consistent action trajectories for both specialist and generalist policies.  
[Link](https://huggingface.co/papers/2603.10126)

**AstraFlow: Dataflow-Oriented Reinforcement Learning for Agentic LLMs**  
Decouples rollout services, dataflow management, and training into autonomous components for agentic RL, enabling efficient multi-policy training on heterogeneous cross-region compute. Achieves comparable accuracy to existing RL systems while speeding up training time by 2.7x.  
[Link](https://huggingface.co/papers/2605.15565)

**Evaluating Cognitive Age Alignment in Interactive AI Agents**  
Introduces ChildAgentEval, the first psychometric benchmark comparing MLLM-based agent reasoning against age-specific human developmental stages, exposing fundamental gaps where state-of-the-art agents fail at tasks a child can resolve with ease.  
[Link](https://huggingface.co/papers/2605.17894)

**MementoGUI: Learning Agentic Multimodal Memory Control for Long-Horizon GUI Agents**  
Proposes MementoGUI, a plug-in memory framework equipping GUI agents with MementoCore — a learned controller for online memory selection, compression, and retrieval — enabling long-horizon task state maintenance without finetuning the backbone agent.  
[Link](https://huggingface.co/papers/2605.18652)

**CHI-Bench: Can AI Agents Automate End-to-End, Long-Horizon, Policy-Rich Healthcare Workflows?**  
A benchmark of healthcare workflows (prior authorization, utilization management, care management) exposing massive agent gaps: best agent resolves only 28% of tasks with no agent clearing 20% on strict criteria, and single-session execution drops to 3.8%, suggesting similar deficits in other policy-dense enterprise domains.  
[Link](https://huggingface.co/papers/2605.16679)

**VideoSeeker: Incentivizing Instance-level Video Understanding via Native Agentic Tool Invocation**  
Proposes a paradigm for instance-level video understanding through visual prompts, integrating agentic reasoning with proactive visual perception. Achieves +13.7% improvement over baselines and surpasses GPT-4o and Gemini-2.5-Pro on spatiotemporal localization tasks.  
[Link](https://huggingface.co/papers/2605.16079)

**E-PMQ: Expert-Guided Post-Merge Quantization with Merged-Weight Anchoring**  
Addresses post-merge quantization (PMQ) by using source expert weights as calibration guides and merged-weight anchoring, enabling low-bit quantization of multi-expert merged models. Improves 4-bit GPTQ from 65% to 73.6% on CLIP-ViT-B and from 34.8% to 76.7% on harder 20-task settings.  
[Link](https://huggingface.co/papers/2605.16882)

**TOBench: A Task-Oriented Omni-Modal Benchmark for Real-World Tool-Using Agents**  
A benchmark of 100 executable tasks (Customer Service, Intelligent Creation) using 27 MCP servers and 324 tools, evaluating agents on closed-loop multimodal verification with self-correction. Claude Opus 4.6 achieves only 32% task success vs. 94% human benchmark.  
[Link](https://huggingface.co/papers/2605.16909)

**NGM: A Plug-and-Play Training-Free Memory Module for LLMs**  
N-gram Memory (NGM) uses a Causal N-Gram Encoder and Cosine-Gated Memory Injector to provide training-free memory augmentation for LLMs with no additional memory table or retrieval pipeline. Improves Qwen3 by 0.5–1.2 points on average, with +3.0 on LiveCodeBench and +3.03 on GPQA.  
[Link](https://huggingface.co/papers/2605.16893)
