# Research Debrief — June 03, 2026 04:00

**0 arXiv papers** · **44 web findings**

## ⚡ Quick Summary

- On-Policy Distillation breakthrough: TrOPD introduces trust-region-based OPD handling teacher-student distribution mismatch through credit assignment strategies, outperforming all prior OPD methods across math, code, and general-domain benchmarks.
- Whole-body motion tracking at scale: Humanoid-GPT trains a GPT-style Transformer on 2B frames of unified motion data, achieving unprecedented zero-shot generalization to unseen motions and control tasks.
- KV cache eviction for reasoning models: VaSE protects large-magnitude value states and uses stochastic eviction, bridging the accuracy gap between eviction and selection-based sparse attention for 4x compression.
- Visual future simulation for reasoning: PF-OPSD combines world model rollouts with MLLM abstract reasoning, outperforming baselines by 10.6% on spatial lookahead tasks and increasing robustness to noisy rollouts.

- AI agents reshape computing: Microsoft CEO Satya Nadella signals a fundamental shift from OS-and-apps to agent-first computing, reorganizing core products around AI agent workflows.
- VSCode GitHub token theft vulnerability: A 1-click attack allows stealing GitHub tokens via a VSCode bug, exposing private repositories to unauthorized read/write access.
- HTTP/2 bomb rediscovered: A 14-year-old HTTP header compression vulnerability was rediscovered and published, affecting servers that use HTTP/2 compression.
- NVIDIA generative world model for AV simulation: OmniDreams uses Cosmos diffusion for photorealistic action-conditioned video generation, enabling closed-loop autonomous driving policy evaluation at scale.
- ChatGPT+Cloudflare privacy analysis: Technical analysis reveals how ChatGPT blocks typing until Cloudflare reads React state, raising questions about client-side surveillance in AI interfaces.
- AI for sustainable development: A Nature paper investigates how AI is being applied to UN Sustainable Development Goals, mapping research foci and emerging opportunities.
