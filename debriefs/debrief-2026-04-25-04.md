## ⚡ Quick Summary

- **LVLMs hallucinate significantly when natural-language prompts override visual grounding (HalluScope benchmark quantifies this failure mode), and agents can now convert research questions into full scientific workflows end-to-end via a 3-layer semantic architecture.**
- **LoRA at the crossroads: GiVA achieves LoRA-matching performance at 8x lower rank using gradient-informed vector initialization, while LoRA-Redux provides a comprehensive signal-processing analysis of 40+ adapter variants for large models.**
- **Multi-commit Python vulnerabilities (CrossCommitVuln-Bench) show each commit is individually invisible to per-commit static analysis, exposing fundamental limits of tools like Semgrep and Bandit on real-world CVE-level bugs.**
- **A new multi-turn LLM attack called Transient Turn Injection (TTI) distributes adversarial intent across isolated stateless turns, systematically bypassing policy enforcement in both commercial and open-source models.**
- **Omni multimodal model demonstrates Context Unrolling — reasoning explicitly across text, images, video, 3D geometry, and hidden representations before predicting, enabling more faithful aggregation of heterogeneous multimodal knowledge.**
