⚡ Quick Summary

  • Autonomous AI agents (Codex/Claude) ran 10,000 nanoGPT optimization attempts over two weeks, setting a new 2,930-step record and revealing critical failure modes in self-directed research loops.
  • SU-01 reasoning model reaches IMO gold-medal level on a 30B-A3B backbone using reverse-perplexity curriculum SFT and two-stage RL, with stable trajectories exceeding 100K tokens.
  • Orchard open-source agentic framework achieves 67.5% on SWE-bench (new open-source SOTA) and 74.1% on WebVoyager with a 4B vision-language computer-use agent trained on just 2.2K tasks.
  • Darwin evolutionary merging produces a 27B model ranking #6 of 1,252 models on GPQA Diamond without any gradient training, using MRI-Trust fusion across 14-dimensional merge genomes.
  • WildClawBench reveals frontier agents (Claude Opus 4.7 best at 62.2%) still struggle with long-horizon native-runtime CLI tasks averaging 8 minutes and 20+ tool calls.

📰 News Headlines

  • AI-assisted CVE discovery is flooding disclosure volumes: A VulnCheck analysis shows public CVE disclosures are surging, with AI-assisted vulnerability discovery increasingly identified as the primary driver.
  • Linux kernel 0-day exposes /etc/shadow to unprivileged users: A new ptrace-based exploit allows stealing SSH host keys and shadow password files on unpatched kernels.
  • Mullvad VPN server fleet enables user fingerprinting: With only 578 servers versus Proton VPN's 20,000, Mullvad's per-user exit IP assignment creates a trackable fingerprint across sites.
  • ChatGPT client-side Cloudflare gate reads React state before input is allowed: Reverse-engineering reveals Cloudflare middleware intercepts and inspects ChatGPT's internal React component state.
  • Vibe coding eroding programmers' edge-case reasoning: A widely-discussed essay argues AI-assisted bug handling is silently degrading developers' ability to reason through complex code paths.
