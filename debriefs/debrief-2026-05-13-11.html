⚡ Quick Summary

  • RL unlocks parametric knowledge in LLMs — Reinforcement learning achieves ~27% relative gains on factual QA by redistributing probability mass over existing knowledge rather than acquiring new facts, broadening RL's role beyond reasoning alone.
  • Multi-Stream LLMs unblock sequential bottlenecks — New paradigm switches from sequential chat formats to parallel streams of thoughts, inputs and outputs, enabling models to think while reading or act while writing.
  • Agent-BRACE introduces belief-state representations — Decouples LLM agents into belief state and policy models with verbalized uncertainty labels, improving long-horizon partially observable tasks.
  • SenseNova-U1 unifies multimodal understanding and generation — NEO-unify architecture breaks the traditional dichotomy between understanding and generation, rivaling top-tier VLMs while delivering native any-to-image synthesis.
  • Continual Harness enables reset-free self-improving agents — Formalizes online self-improvement for embodied agents without human-in-the-loop, substantially reducing button-press cost on Pokemon benchmarks.

📰 News Headlines

  • Google finds first AI-developed zero-day bypassing 2FA — Google security researchers identified the first genuine zero-day exploit actually developed by an AI system, crossing a significant threshold in AI-driven cyberattacks.
  • AI startup Grego prevents $27.7M financial attack — A novel AI reasoning system detected and blocked a massive fraud attempt, demonstrating the emerging commercial value of advanced AI security applications.
  • YellowKey Bitlocker bypass vulnerability disclosed — A new attack vector against Windows Bitlocker encryption was publicly released, enabling encrypted drive access under specific conditions.
  • Six critical CVEs disclosed in dnsmasq — Multiple serious vulnerabilities in the widely-used dnsmasq DNS forwarder affect millions of networked devices.
  • Claude Code RCE via deeplink settings injection — A remote code execution vulnerability in Claude Code was found and patched, exploiting how the tool handles settings injection through deeplink handlers.
