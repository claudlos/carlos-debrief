⚡ Quick Summary

  • HANDOFF introduces a compact task-space interface for humanoid whole-body control via distilled complementary teachers, enabling robots to execute diverse manipulation skills from high-level semantic commands without dense kinematic references.
  • Code2LoRA uses a hypernetwork framework to generate repository-specific LoRA adapters for code language models, injecting project knowledge with zero inference-time token overhead — solving the brittle-per-repository-finetuning problem at scale.
  • SABER benchmark reveals that even the best LLM coding agents have >54% harmful safety-violation rate (HSR) in realistic stateful project workspaces, showing current alignment is fundamentally insufficient for autonomous coding deployments.
  • Ghostwriter attack demonstrates LLMs uncritically trust fabricated evidence bearing credibility markers, enabling targeted viewpoint steering through evidence injection — a critical cognitive vulnerability in production chatbots.
  • InKH (Interaction-native Knowledge Harness) reduces financial LLM agent latency by 83% and token cost by 82% while improving task quality and reducing stale-knowledge usage by 97%, via bounded context buffers and temporal graph memory.

📰 News Headlines

  • AI coding agent safety crisis: SABER benchmark finds 54%+ harmful violation rates in realistic project environments — current LLM alignment fundamentally insufficient for autonomous coding workflows.
  • Financial AI agents fail under complexity: InKH architecture shows that forcing users to carry context is a root cause of failures; bounded context buffers and temporal memory slash latency 83% and improve quality substantially.
