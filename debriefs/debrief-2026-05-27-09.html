⚡ Quick Summary

  • D²-Monitor: A bi-level safety monitor for diffusion LLMs that detects when lightweight probes struggle via hidden-state 'hesitation' signals, routing to heavier probes only when needed — achieving SOTA safety monitoring with only 0.85M parameters.
  • Collaborative Parallel Thinking (CPT): A training-free inference trick that lets parallel test-time search branches share intermediate discoveries via a deduplicated information pool, improving accuracy-latency trade-off on math benchmarks.
  • DarkForest: A controlled-communication multi-agent LLM framework — keeps agents independent, clusters semantic equivalents, and computes calibrated beliefs over clusters — improving accuracy up to 30.7% while cutting token use 6.5x.
  • MiniMax-M2 series: A 229.9B MoE LLM (9.8B active) with an agent-native RL system called Forge, designed for agentic coding and deep search; M2.7 autonomously debugs its own training runs.
  • Personalized Memory Benchmark: PerMemBench evaluates personalized memory in LLM agents across multi-year interaction histories — finding personalization yields large gains but accurate gating of what to store per-user remains open.

📰 News Headlines

  • Keystroke-level LLM telemetry tracking: A security researcher reverse-engineered how ChatGPT/Cloudflare tracks React state server-side before every user keystroke — a mechanism that clones full session state on prompt submit.
  • Post-quantum IoT risk: A Springer survey calls for urgent migration of IoT devices to post-quantum cryptography before quantum computers can break today's encryption — identified as a strategic infrastructure risk.
  • LLMs in computational social science: A comprehensive survey examines how large language models are reshaping social science workflows, mapping the landscape of new capabilities alongside challenges of data bias and privacy.
  • Long-form AI video evaluation matures: EvalVerse and LongAV-Compass represent purpose-built evaluation frameworks for minute-scale AI-generated video, assessing cinematic quality, acting, and audio-visual consistency beyond simple prompt-following.
  • MoE architecture reaches commodity smartphones: MobileMoE delivers 1.8–3.8x speedup over dense baselines at INT4 memory on standard smartphones, establishing a new Pareto frontier for on-device inference on sub-billion parameter models.
