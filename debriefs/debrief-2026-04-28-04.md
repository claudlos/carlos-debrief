## Quick Summary

- **Learning with Chain-of-Thought from Multiple Thinkers**: Studies how models learn from multiple correct CoT solutions — computationally easy with CoT but hard with outcome-only supervision, revealing systematic differences in how thinkers solve problems.
- **Adaptive Runtime Governance for Autonomous AI Agents**: Proposes the Informational Viability Principle — bounding unobserved risk via U(x)+SB(x)+RG(x) — and allowing actions only when capacity exceeds estimated risk by a safety margin.
- **Green Shielding: Benign Input Variation Breaks AI Trustworthiness**: Shows LLM outputs are highly sensitive to routine non-adversarial query phrasing; proposes CUE criteria (Context, Use-case, Evidence) for evidence-backed deployment guidance.
- **Defective Task Descriptions Undermine LLM Code Generation**: Detects task description defects in LLM code generation using a lightweight classifier; finds vague or malformed requirements strongly affect code correctness.
- **Clinical AI Evaluation via Case-Specific Clinician-Authored Rubrics**: Shows LLM-generated rubrics approximate clinician agreement across 823 encounters, offering a scalable clinically valid, economically viable AI evaluation methodology.
