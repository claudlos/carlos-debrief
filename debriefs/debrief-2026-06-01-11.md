### HuggingFace Papers

**1. GrepSeek: Training Search Agents for Direct Corpus Interaction**
- **Authors**: Alireza Salemi, Chang Zeng, Atharva Nijasure, et al. (71 upvotes)
- **What it does**: Trains a compact LLM agent to find, filter, and compose evidence from large text corpora using executable shell commands (grep, find, awk) rather than keyword-based retrievers. Uses a two-stage pipeline (cold-start Tutor+Planner, then GRPO refinement) with a sharded-parallel execution engine.
- **Why it matters**: Complements traditional RAG by treating the corpus itself as the search environment. Achieves 7.6x speedup via parallel shell execution while preserving byte-exact results — enabling practical large-scale direct corpus interaction.
- [Link](https://huggingface.co/papers/2605.29307)

**2. Trust-Region Behavior Blending for On-Policy Distillation**
- **Authors**: Daniil Plyusov, Alexey Gorbatovski, Alexey Malakhov, et al. (43 upvotes)
- **What it does**: Addresses the problem of weak/low-quality early student rollouts polluting teacher supervision during on-policy distillation (OPD). TRB replaces the early rollout policy with the closest-to-teacher behavior inside a KL trust region, annealing the budget to zero as training progresses.
- **Why it matters**: Warmup method significantly improves OPD results on math-reasoning distillation tasks by preventing the student from learning from its own poor early outputs — more principled than ad-hoc tricks.
- [Link](https://huggingface.co/papers/2605.31159)

**3. Mellum2 Technical Report**
- **Authors**: Marko Kojic, Ivan Bondyrev, Aral de Moor, et al. (31 upvotes)
- **What it does**: An open-weight 12B-parameter MoE language model (2.5B active params/token) specialized in software engineering: code generation/editing, debugging, multi-step reasoning, tool use, function calling, and agentic coding. Features YaRN 128K context extension and dual Instruct/Thinking variants.
- **Why it matters**: Strong open alternative for coding agents — combines MoE efficiency with multi-token prediction (auxiliary pre-training + speculative draft), RLVR post-training, and FP8 Muon optimization for commodity GPU deployment.
- [Link](https://huggingface.co/papers/2605.31268)

**4. DecMem: Towards Minute-Long Consistent World Generation with Decoupled Memory**
- **Authors**: Zhenhao Yang, Xiaoshi Wu, Zhengyao Lv, et al. (8 upvotes)
- **What it does**: A decoupled memory architecture for video world models — Sparse Global Memory for efficient fine-grained access to global history plus Anchored Local Memory for stable high-quality extrapolation. Solves attention dispersion and computational inefficiency in long-horizon video generation.
- **Why it matters**: Breaks the minute-long consistent video generation barrier using learnable memory rather than explicit 3D or coarse frame modeling. State-of-the-art on long-horizon fidelity and temporal consistency.
- [Link](https://huggingface.co/papers/2605.31336)

**5. How can embedding models bind concepts?**
- **Authors**: Arnas Uselis, Darina Koishigarina, Seong Joon Oh (4 upvotes)
- **What it does**: Studies why CLIP struggles with concept binding (determining which color belongs to which shape in multi-object scenes) despite recognizing individual concepts well. Finds scene embeddings decompose additively into object representations, and that CLIP's binding function is high-complexity — preventing generalization.
- **Why it matters**: Theoretically isolates why current VLMs fail at binding and demonstrates in controlled models that low-complexity multiplicative binding functions enable systematic generalization — a path toward robust scene understanding.
- [Link](https://huggingface.co/papers/2605.31503)

**6. VisualThink-VLA: Visual Intermediate Reasoning for Effective and Low-Latency VLA Policies**
- **Authors**: Mingjian Gao, Wenqiao Zhang, Yuqian Yuan, et al. (2 upvotes)
- **What it does**: Bootstraps vision-language-action policy prediction through a compact visual-evidence interface that avoids textual chain-of-thought decoding overhead. Uses selective routing to learn visual evidence tokens and introduces VisualEvidence-Kit (754.7k instructions) for supervision.
- **Why it matters**: Reduces VLA step latency from 8.4s to 0.37s (22.8x speedup) on BridgeData V2 while achieving the highest success rates across benchmarks — bringing real-time embodied control closer to viability.
- [Link](https://huggingface.co/papers/2605.30011)

**7. GDSD: Reinforcement Learning as Guided Denoiser Self-Distillation for Diffusion Language Models**
- **Authors**: Xiaohang Tang, Keyue Jiang, Che Liu, et al. (3 upvotes)
- **What it does**: Replaces the ELBO likelihood surrogate in RL for diffusion LLMs with direct denoiser self-distillation from an advantage-guided teacher, derived from closed-form optimum of reverse-KL regularized RL. Matches dLLM denoiser logits to teacher via normalization-free objective.
- **Why it matters**: Eliminates training-inference mismatch bias from ELBO surrogates. Up to +19.6% test-accuracy improvement on planning, math, and coding benchmarks with LLaDA-8B and Dream-7B, with more stable training reward dynamics.
- [Link](https://huggingface.co/papers/2605.29398)

**8. SAAS: Self-Aware RL for Over-Search Mitigation in Agentic Search**
- **Authors**: Yunbo Tang, Chengyi Yang, Shiyu Liu, et al. (14 upvotes)
- **What it does**: Prevents agents from blindly triggering unnecessary searches and failing to terminate when evidence is sufficient. Uses search boundary modeling via contrastive rollouts, boundary-aware trajectory penalties, and stage-wise curriculum that prioritizes reasoning over search regularization.
- **Why it matters**: Directly addresses the over-search problem that makes agentic LLMs prohibitively expensive in production — substantially reduces inference cost while maintaining accuracy.
- [Link](https://huggingface.co/papers/2605.29796)

**9. From Model Scaling to System Scaling: Scaling the Harness in Agentic AI**
- **Authors**: Shangding Gu (1 upvote)
- **What it does**: Argues that the next major bottleneck in agentic AI is the agent harness — the structured execution layer around foundation models (memory, context, skill routing, orchestration, verification, governance) — not model scaling. Studies three core harness bottlenecks: context governance, trustworthy memory, dynamic skill routing.
- **Why it matters**: Reframes agent evaluation beyond one-shot task success to trajectory quality, memory hygiene, and context efficiency. Proposes a research agenda for harness-level benchmarks as a new first-class design object.
- [Link](https://huggingface.co/papers/2605.26112)

**10. One Click per Cell Type Suffices: Training-free Group Interaction for Cell Instance Segmentation**
- **Authors**: Sanghyun Jo, Seo Jin Lee, Seohyung Hong, et al. (1 upvote)
- **What it does**: Chain-of-Prompts (CoP) shifts interactive segmentation from O(N) per-instance to O(T) per-cell-type using a single click. Exploits the fact that SAM's frozen encoder already clusters same-type cells — recursively expands clicks via non-parametric gating and spatially distant point selection.
- **Why it matters**: One click per type retains over 90% of per-instance performance with no additional training. Surpasses fully supervised methods on three benchmarks and achieves 99%+ on morphologically homogeneous datasets — dramatically reduces annotation cost.
- [Link](https://huggingface.co/papers/2605.29429)

**11. When Confidence Misleads: Suffix Anchoring for Diffusion Language Models**
- **Authors**: Jungwon Park, Jimyeong Kim, Jungmin Ko, et al. (2 upvotes)
- **What it does**: Studies why confidence misleads fully non-autoregressive diffusion language model decoding — EOT tokens receive high confidence causing incomplete generation, while suffix anchoring causes local overconfidence near the anchor. Proposes Suffix-Anchored Confidence Modulation.
- **Why it matters**: Preserves parallel decoding advantage of fully non-AR generation while improving response completion and reducing premature token decoding near anchors — consistent improvements across text, vision-language, and code benchmarks.
- [Link](https://huggingface.co/papers/2605.28181)

**12. HL-OutPaint: Coarse-to-Fine Video Outpainting for High-Resolution Long-Range Videos**
- **Authors**: Jeongeun Park, Janghyeok Han, Geonung Kim, et al. (2 upvotes)
- **What it does**: High-resolution video outpainting via coarse-to-fine pipeline: Global Coarse Guidance (GCG) built from global-local frame swapping captures global structure and dominant motion, then guides high-resolution outpainting for spatially detailed, temporally consistent content.
- **Why it matters**: Achieves stable, coherent generation for large spatial extrapolation and long sequences — addressing a key limitation in adapting videos to diverse display formats.
- [Link](https://huggingface.co/papers/2605.17543)

**13. Memory-Bound but Not Bandwidth-Limited: Physical AI Inference Gap**
- **Authors**: Josef Chen
- **What it does**: Measures batch-1 decode for 7-8B GQA transformers across H100, A100, L40S, and L4 at context lengths 2048-16384. Finds that faster memory does not proportionally translate to latency gains. Identifies a launch-side overhead (exposed by CUDA Graphs A/B testing) that becomes visible on fast GPUs but hidden on slower ones.
- **Why it matters**: The common "memory-bandwidth-bound" description of physical AI inference is incomplete. Deployment on fast GPUs needs CUDA Graphs or similar to achieve expected performance — the launch overhead is a significant hidden bottleneck.
- [Link](https://huggingface.co/papers/2605.30571)

**14. Flat-Pack Bench: Evaluating Spatio-Temporal Understanding in LVLMs**
- **Authors**: Aditya Chetan, Eric Cai, Peeyush Kushwaha, et al. (3 upvotes)
- **What it does**: Benchmark for LVLMs on furniture assembly tasks: temporal ordering of assembly actions, localization of assembly state, understanding part mating, and tracking. Uses multiple-choice questions with visual prompts highlighting relevant parts.
- **Why it matters**: Reveals that state-of-the-art LVLMs struggle significantly with fine-grained spatio-temporal reasoning — limited temporal information utilization, weak tracking ability, poor understanding of physical contact. Identifies key gaps for future embodied AI research.
- [Link](https://huggingface.co/papers/2605.21625)

**15. TSCOMP: Component-level Benchmarking for Deep Multivariate Time-Series Forecasting**
- **Authors**: Shuang Liang, Chaochuan Hou, Xu Yao, et al.
- **What it does**: First large-scale benchmark systematically deconstructing forecasting methods into fine-grained components (preprocessing, encoding, network architectures, optimization). Uses constrained orthogonal experimental design with 20,000+ model-dataset evaluations.
- **Why it matters**: Corpus-driven automated component selection outperforms manually designed complex architectures — validates that systematic component selection (not just end-to-end tuning) is the path forward for time-series forecasting.
- [Link](https://huggingface.co/papers/2605.26562)

**16. A Topology-Aware Spatiotemporal Handover Framework for Multi-UAV Tracking**
- **Authors**: Jianlin Ye, Christos Kyrkou, Panayiotis Kolios
- **What it does**: Real-time multi-camera multi-vehicle tracking system using deterministic queue-based matching via geometric overlaps and virtual lane discretization for identity handover. Achieves 99.8% Handover Success Rate vs 74.1% for Re-ID baselines on urban traffic flows.
- **Why it matters**: Solves trajectory fragmentation in multi-UAV fields of view — enabling network-level analysis (Origin-Destination estimation) previously impossible with isolated single-drone systems. Validates edge deployment feasibility.
- [Link](https://huggingface.co/papers/2605.15779)
