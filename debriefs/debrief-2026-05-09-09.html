⚡ Quick Summary

  • Recursive Agent Optimization (RAO) introduces a reinforcement learning approach for training agents that recursively spawn sub-task agents, implementing inference-time scaling via divide-and-conquer that naturally generalizes to longer contexts and harder problems.
  • EMO demonstrates that Mixture-of-Experts models pretrained with flexible expert routing develop emergent modular structure, enabling narrow capability subsets without incurring full-model overhead — a significant step toward efficient specialized LLMs.
  • PACZero achieves PAC-private LLM fine-tuning via sign quantization, bounding membership-inference attack success rate to near-random chance ($I(S^*;Y_{1:T})=0$) while maintaining usable model utility.
  • Global LLM Leaderboards are Misleading: analysis of 89K pairwise comparisons across 52 models reveals the best-fit global Bradley-Terry model is outperformed by small, task-specific model subsets, questioning the universality of current ranking methodologies.
  • Benchmarkless Comparative LLM Safety Scoring formalizes a novel framework for validating safety differences between language models without any ground-truth labels, enabling safety comparisons in low-resource languages and regulatory regimes where benchmarks don't yet exist.

📰 News Headlines

  • Linux Kernel io_uring ZCRX Exploit: Security researchers publish a local privilege escalation attack exploiting the io_uring kernel subsystem's ZCRX freelist management, demonstrating root access from an unprivileged u32 — a reminder that kernel attack surfaces remain active even in mature subsystems.
  • Cloudflare Intercepts ChatGPT Prompts at Client-Side: A security researcher dissects how ChatGPT's Cloudflare integration reads React component state before the user's prompt reaches OpenAI's servers, raising fresh questions about client-side prompt injection and third-party middleware interception.
  • Per-Function killswitch Merged into Linux Kernel: A new kernel mitigation primitive allows individual functions to be short-circuited at runtime, providing a fine-grained mechanism for security hardening that can disable buggy or vulnerable code paths without full system recompilation.
  • Post-Quantum Cryptography for IoT Authentication: Research reviews the landscape of NIST-standardized post-quantum algorithms (CRYSTALS-Kyber, Dilithium, Falcon) for resource-constrained IoT devices, finding hybrid classical-PQC schemes currently offer the most practical path to quantum-resistant authentication.
  • Quantum Machine Learning Integration Accelerating: A comprehensive review finds growing convergence between quantum algorithms and classical ML techniques, with quantum-speedup demonstrated in optimization, drug discovery, and quantum-secured communication applications — though near-term practical advantage remains limited.
