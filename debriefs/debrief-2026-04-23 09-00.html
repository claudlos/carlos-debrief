⚡ Quick Summary

  • AI Security Evaluation: AVISE provides the first modular open-source framework for systematically identifying vulnerabilities and evaluating security across AI systems and models.
  • LLMs Cannot Reason Scientifically: A 25,000+ agent-run study finds LLM-based scientific agents ignore evidence in 68% of traces and fail to exhibit the epistemic patterns that make science self-correcting.
  • Speech Paralinguistic Benchmark: SpeechParaling-Bench expands coverage from under 50 to over 100 fine-grained paralinguistic features for evaluating Large Audio-Language Models across English-Chinese parallel speech.
  • Convergent Number Representations: Language models trained independently learn similar periodic number features (period T=2,5,10), revealing two-tiered geometric structure across Transformers, RNNs, and word embeddings.
  • FIDO2/Passkey Attack Analysis: A systematic study of 890,000+ phishing attacks reveals new vectors against FIDO2 authentication even as passkeys are widely deployed.

📰 News Headlines

  • AI Agents Execute Workflows, Not Science: Across 25,000+ LLM agent runs, evidence is ignored in 68% of traces and refutation-driven belief revision occurs only 26% of the time — current agents lack epistemic self-correction.
  • North Korea Lazarus Group Uses AI to Industrialize Developer Attacks: Expel's deep-dive into state-sponsored actor tradecraft reveals large-scale AI-assisted campaigns targeting open-source developers.
  • Cloudflare Reads ChatGPT React State Before User Input: A developer decrypted the program enabling Cloudflare to access client-side state before keystrokes reach OpenAI, raising privacy concerns.
  • Coding Agents Still Inefficient in the Wild: SWE-chat (6,000 real sessions, 63K user prompts) finds only 44% of agent code survives into user commits and AI-authored code introduces more security vulnerabilities than human-written code.
  • Test-Time Compute Scales Coding Agents: Claude 4.5 Opus on SWE-Bench Verified jumps 70.9% → 77.6% usingRecursive Tournament Voting over rollout trajectory summaries; DR-Venus shows 4B deep research agents are surprisingly viable with only 10K open-data.
