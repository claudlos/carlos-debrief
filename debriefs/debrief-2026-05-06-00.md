## 📄 ArXiv Papers — LLMs

### [13] Counting as a minimal probe of language model reliability](https://huggingface.co/papers/2605.02028)
*by Tianxiang Dai, Jonathan Fan*
`arXiv:2605.02028 | LLMs`

We introduce Stable Counting Capacity, an assay in which models count repeated symbols until failure. Across more than 100 model variants, stable counting capacity remains far below advertised context limits. Model behavior is consistent neither with open-ended logic nor with stable application of a learned rule, but instead with use of a finite set of count-like internal states. Once this resource is exhausted, exact execution collapses into guessing. Fluent performance does not guarantee general, reliable rule following.

### [14] Hallucinations Undermine Trust; Metacognition is a Way Forward](https://huggingface.co/papers/2605.01428)
*by Gal Yona, Mor Geva, Yossi Matias*
`arXiv:2605.01428 | LLMs`

We argue that most factuality gains have come from expanding the model's knowledge boundary rather than improving awareness of that boundary. We propose faithful uncertainty: aligning linguistic uncertainty with intrinsic uncertainty. This is one facet of metacognition -- the ability to be aware of one's own uncertainty and act on it. Metacognition is essential for LLMs to be trustworthy.

### [15] BlenderRAG: High-Fidelity 3D Object Generation via Retrieval-Augmented Code Synthesis](https://huggingface.co/papers/2605.00632)
*by Massimo Rondelli, Francesco Pivi, Maurizio Gabbrielli*
`arXiv:2605.00632 | LLMs`

We present BlenderRAG, a retrieval-augmented generation system operating on a curated multimodal dataset of 500 expert-validated examples across 50 object categories. By retrieving semantically similar examples during generation, BlenderRAG improves compilation success rates from 40.8% to 70.0% and semantic normalized alignment from 0.41 to 0.77 (CLIP similarity) across four state-of-the-art LLMs.

### [17] Talker-T2AV: Joint Talking Audio-Video Generation with Autoregressive Diffusion Modeling](https://huggingface.co/papers/2604.23586)
*by Zhen Ye, Xu Tan, Aoxiong Yin, Hongzhan Lin, Guangyan Zhang*
`arXiv:2604.23586 | LLMs`

We propose Talker-T2AV, an autoregressive diffusion framework where high-level cross-modal modeling occurs in a shared backbone while low-level refinement uses modality-specific decoders. A shared autoregressive language model jointly reasons over audio and video in a unified patch-level token space. Experiments on talking portrait benchmarks show Talker-T2AV outperforms dual-branch baselines in lip-sync accuracy, video quality, and audio quality.

## 📄 ArXiv Papers — Agents

### [8] HiL-Bench (Human-in-Loop Benchmark): Do Agents Know When to Ask for Help?](https://huggingface.co/papers/2604.09408)
*by Mohamed Elfeki, Tu Trinh, Kelvin Luu, Guangze Luo, Nathan Hunt*
`arXiv:2604.09408 | Agents`

Frontier coding agents solve complex tasks when given complete context but collapse when specifications are incomplete. HiL-Bench (Human-in-the-Loop Benchmark) measures selective escalation skill with human-validated blockers. Ask-F1 captures the tension between over-asking and silent guessing. Evaluation across SWE and text-to-SQL domains reveals a large universal judgment gap: no frontier model recovers more than a fraction of its full-information performance when deciding whether to ask.

### [9] Agentic AI Systems Should Be Designed as Marginal Token Allocators](https://huggingface.co/papers/2605.01214)
*by Siqi Zhu*
`arXiv:2605.01214 | Agents`

This position paper argues that agentic AI systems should be designed and evaluated as marginal token allocation economies rather than as text generators priced by the unit. Following a single request through four economic layers, we show all four solve the same first-order condition: marginal benefit equals marginal cost plus latency cost plus risk cost. Adopting marginal token allocation as the shared accounting object predicts recurring failure modes and points to a research agenda in token-aware evaluation, autonomy pricing, congestion-priced serving, and risk-adjusted RL budgeting.

## 📄 ArXiv Papers — AI Safety

### [16] Prior-Aligned Data Cleaning for Tabular Foundation Models](https://huggingface.co/papers/2604.25154)
*by Laure Berti-Equille*
`arXiv:2604.25154 | AI Safety`

We introduce L2C2, the first deep RL framework framing tabular data cleaning as prior alignment: a learned policy sequences operators to minimize the distributional gap between dirty input and the TFM's synthetic prior. Parameterized cleaning actions improve best-found pipeline reward on 9/10 datasets.

## 📄 ArXiv Papers — Machine Learning

### [7] Linking spatial biology and clinical histology via Haiku](https://huggingface.co/papers/2605.00925)
*by Yan Cui, Jacob S. Leiby, Wenhui Lei, Dokyoon Kim, Yanxiang Deng*
`arXiv:2605.00925 | Machine Learning`

Here we present Haiku, a tri-modal contrastive learning model trained on multiplexed immunofluorescence (mIF). It comprises 26.7 million spatial proteomics patches from 3,218 tissue sections across 1,606 patients spanning 11 organ types, with matched hematoxylin and eosin (H&E) histology and clinical metadata aligned in a shared embedding space. Haiku enables three-way cross-modal retrieval, improves downstream classification and clinical prediction tasks over unimodal baselines, and supports zero-shot biomarker inference.

### [10] Assessing Pancreatic Ductal Adenocarcinoma Vascular Invasion: the PDACVI Benchmark](https://huggingface.co/papers/2604.27582)
*by M. Riera-Marin, O. K. Sikha, J. Rodriguez-Comas, M. S. May, T. Kirscher*
`arXiv:2604.27582 | Machine Learning`

We introduce CURVAS-PDACVI, an open benchmark for uncertainty-aware AI in PDAC staging based on a densely annotated dataset with five independent expert annotations per scan. We propose a multi-metric evaluation framework extending beyond spatial overlap to include probabilistic calibration. Evaluation shows that methods optimized for binary segmentation often degrade in high-complexity cases, whereas methods modeling inter-rater disagreement produce better calibrated probabilistic maps.

### [11] A Hybrid Approach for Closing the Sim2real Appearance Gap in Game Engine Synthetic Datasets](https://huggingface.co/papers/2605.02291)
*by Stefanos Pasios*
`arXiv:2605.02291 | Machine Learning`

This letter investigates using FLUX.2-4B Klein and REGEN to enhance photorealism of game engine synthetic datasets. We propose a hybrid approach combining diffusion-based geometry and material transformations with the distribution-matching of image-to-image translation. Experiments show REGEN outperforms FLUX.2-4B Klein alone, and combining both achieves better visual realism than either individually while maintaining semantic consistency.

### [12] Linear-Time Global Visual Modeling without Explicit Attention](https://huggingface.co/papers/2605.01711)
*by Ruize He, Dongchen Han, Gao Huang*
`arXiv:2605.01711 | Machine Learning`

We demonstrate that attention can be mathematically reframed as a Multi-Layer Perceptron (MLP) equipped with dynamically predicted parameters. We investigate whether Transformer-level sequence global modeling can be achieved entirely through dynamic parameterization while maintaining linear complexity. Extensive empirical studies on vision models demonstrate that dynamic parameterization serves as a highly effective, linear-complexity alternative to explicit attention.
