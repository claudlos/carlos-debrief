## ⚡ Quick Summary

- <strong>FutureSim</strong> — Grounded world-event simulation where AI agents forecast real news beyond their knowledge cutoff by replaying chronological events, enabling evaluation of adaptive reasoning in open-ended environments.
- <strong>OpenDeepThink</strong> — Population-based test-time compute framework using Bradley-Terry pairwise aggregation to scale breadth in LLM reasoning, overcoming the single-trace depth bottleneck without ground-truth verifiers.
- <strong>RefDecoder</strong> — Conditional video decoding with equal decoder-side conditioning, addressing detail loss and inconsistency in latent diffusion video generation models.
- <strong>MetaBackdoor</strong> — First backdoor attacks on LLMs exploiting positional encoding as the trigger surface, requiring no content modification and bypassing existing defenses.
- <strong>WARD</strong> — Adversarially robust prompt injection defense for web agents, addressing generalization to unseen domains and adversarial attacks evolving over time.
- <strong>MeMo (Memory as a Model)</strong> — Encodes new domain knowledge into a dedicated memory model without updating LLM parameters, enabling timely specialized information.
