## ArXiv Papers

### 🧠 LLMs

**1. Code2LoRA: Hypernetwork-Generated Adapters for Code Language Models under Software Evolution**  
Authors: Liliana Hotsko, Yinxi Li, Yuntian Deng, Pengyu Nie  
Code language models need repository-level context to resolve imports, APIs, and project conventions. Existing methods inject this knowledge as long inputs via RAG or per-repository fine-tuning — costly at repository scale and brittle to evolving codebases. Code2LoRA introduces a hypernetwork framework that generates repository-specific LoRA adapters, injecting project knowledge with zero inference-time token overhead. After training, the generated adapters can be merged into the base model with no architectural change, enabling seamless deployment across thousands of repositories. This fundamentally solves the scalability and brittleness problems that have held back repository-scale code intelligence.  
`cs.SE, cs.AI, cs.CL` | https://arxiv.org/abs/2606.06492

**2. PAR3D: A Unified 3D-MLLM with Part-Aware Representation for Scene Understanding**  
Authors: Shaohui Dai, Yansong Qu, You Shen, Shengchuan Zhang, Liujuan Cao  
Existing 3D multimodal LLMs (3D-MLLMs) remain largely object-centric, limiting their ability to model fine-grained part structures essential for embodied interaction with 3D environments. PAR3D introduces a unified part-aware 3D-MLLM framework that enables models to understand, segment, and reason about sub-object parts in 3D scenes. The approach uses a part-aware representation that hierarchically decomposes scenes into semantic components, enabling precise spatial reasoning for tasks like part-level visual question answering and referring segmentation. This capability is critical for robots that must manipulate specific components of objects in cluttered real-world scenes.  
`cs.CV` | https://arxiv.org/abs/2606.06485

**3. You Only Index Once: Cross-Layer Sparse Attention with Shared Routing**  
Authors: Yutao Sun, Yanqi Zhang, Li Dong, Jianyong Wang, Furu Wei  
Long-context LLM inference is increasingly bottlenecked by decoding efficiency, especially in reasoning-heavy settings where models generate long chains of thought. Existing sparse attention methods face a quality-efficiency trade-off: structured block methods give strong acceleration but noticeable quality loss, while token-sparse methods are more accurate but deliver limited end-to-end speedup due to top-k routing overhead. YOTO proposes cross-layer sparse attention with shared routing, allowing the model to index once and reuse that decision across multiple attention layers, dramatically reducing routing computation while maintaining accuracy.  
`cs.CL, cs.AI, cs.LG` | https://arxiv.org/abs/2606.06467

### 🤖 Machine Learning

**4. TailLoR: Protecting Principal Components in Parameter-Efficient Continual Learning**  
Authors: Marius Dragoi, Ioana Pintilie, Alexandra Dragomir, Antonio Barbalau, Florin Brad  
Parameter-efficient finetuning methods based on spectral decomposition have enabled progress in Continual Learning, but they still suffer from interference between tasks. TailLoR utilizes the singular bases U and V of pre-trained weights as a fixed reference frame to learn a low-rank update applied to the singular value matrix, with a soft spectral penalty discouraging updates aligned with dominant singular directions. This routes fine-grained adaptation into the flexible long-tail spectrum of singular values, protecting the principal components that encode core task performance. Experiments show significant improvement in catastrophic forgetting across diverse continual learning scenarios.  
`cs.LG` | https://arxiv.org/abs/2606.06494

**5. DNQ: Deep Nash Q-Network for Partially Observable n-Player Games**  
Authors: Qintong Xie, Edward Koh, Xavier Cadet, Peter Chin  
Many real-world competitive systems require multiple decision-makers to act simultaneously under shared constraints and limited information. DNQ proposes a solver-in-the-loop equilibrium supervision framework for training bidding agents in multi-turn simultaneous games, alternating between trajectory collection and critic-based payoff estimation with equilibrium constraints. The approach addresses the fundamental challenge of learning in partially observable environments where each player sees only their own local information and must reason about others' strategies. Experiments on auction and resource allocation domains show DNQ agents significantly outperform prior game-theoretic learning methods.  
`cs.GT, cs.LG` | https://arxiv.org/abs/2606.06480

**6. How abundant are good interpolators?**  
Authors: August Y. Chen, Ahmed El Alaoui  
Let S be the set of unit norm linear classifiers that correctly classify every point of a labeled dataset with a fixed margin κ. Under Gaussian mixture model and logistic model data-generating distributions, in the proportional regime n/d → α with small enough α, this paper establishes a large deviation principle for the probability that a random direction θ ∈ S is a "good interpolator." The result characterizes exactly how rare well-performing interpolating classifiers are as dimension and sample size grow proportionally — with broad implications for the classical bias-variance tradeoff and modern neural network generalization theory.  
`math.ST, cs.LG, math.PR` | https://arxiv.org/abs/2606.06469

**7. RREDCoT: Segment-Level Reward Redistribution for Reasoning Models**  
Authors: Mykyta Ielanskyi, Kajetan Schweighofer, Lukas Aichberger, Sepp Hochreiter  
Recent reasoning language models are driven by RL fine-tuning with GRPO, but final-answer rewards only arrive after the complete chain-of-thought trace — making reward assignment a delayed problem that GRPO handles poorly (Monte Carlo estimation). RREDCoT redistributes rewards at the segment level rather than the full trace level, enabling more granular credit assignment for intermediate reasoning steps. This accelerates RL convergence for reasoning tasks and improves final answer quality, particularly on math and logic benchmarks where multi-step reasoning is essential.  
`cs.LG, cs.AI` | https://arxiv.org/abs/2606.06475

**8. Self-Augmenting Retrieval for Diffusion Language Models**  
Authors: Paul Jünger, Justin Lovelace, Linxi Zhao, Dongyoung Go, Kilian Q. Weinberger  
Diffusion language models generate text by iteratively denoising an entire response in parallel, committing confident token predictions and discarding uncertain ones. This paper shows that discarded tokens are actually a useful lookahead signal for retrieval-augmented generation — even low-confidence tokens surface salient entities early in the denoising trajectory, enabling retrieval of stronger evidence. The self-augmenting approach uses these discarded tokens to dynamically improve retrieval quality during generation, reducing hallucinations and improving factual accuracy at no additional inference cost.  
`cs.CL, cs.AI, cs.LG` | https://arxiv.org/abs/2606.06474

**9. MLEvolve: A Self-Evolving Framework for Automated Machine Learning Algorithm Discovery**  
Authors: Shangheng Du, Xiangchao Yan, Jinxin Shi, Zongsheng Cao, et al.  
Existing MLE (Machine Learning Engineering) agents suffer from inter-branch information isolation, memoryless search, and lack of hierarchical control, which together hinder long-horizon optimization. MLEvolve introduces an LLM-based self-evolving multi-agent framework for end-to-end machine learning algorithm discovery, using a hierarchical control structure that enables persistent memory across search branches and structured exploration-exploitation balancing. The system can iteratively improve its own search strategies based on past successes and failures, achieving new state-of-the-art results on NASBench-style benchmarks and discovering novel architecture patterns that outperform human-designed baselines.  
`cs.AI, cs.CL` | https://arxiv.org/abs/2606.06473

**10. PC Layer: Polynomial Weight Preconditioning for Improving LLM Pre-Training**  
Authors: Senmiao Wang, Tiantian Fang, Haoran Zhang, Yushun Zhang, Kunxiang Zhao, Alex Schwing, Ruoyu Sun  
LLM pre-training suffers from unstable weight conditioning throughout the optimization trajectory, leading to gradient explosion or vanishing that degrades final model quality. PC Layer reshapes the singular-value spectrum of weight matrices via low-degree polynomial preconditioning, ensuring stable weight conditioning throughout training. After training, the preconditioned weights can be merged back into the original architecture, incurring no inference overhead. Demonstrated on Llama-1B pre-training with significant improvement in final model performance and training stability.  
`cs.LG, cs.AI` | https://arxiv.org/abs/2606.06470

**11. Pretraining Recurrent Networks without Recurrence**  
Authors: Akarsh Kumar, Phillip Isola  
Standard backpropagation through time (BPTT) for RNNs is sequential in time, limiting parallelism, and suffers from vanishing or exploding gradients over long sequences. Supervised Memory Training (SMT) reduces RNN training to supervised learning by sidestepping recurrent credit propagation entirely — using a memory mechanism that enables direct credit assignment to distant time steps without temporal unfolding. This enables highly parallel training of RNNs while achieving better long-range association learning than BPTT, potentially bridging the gap between RNNs and transformer-based approaches.  
`cs.LG, cs.AI` | https://arxiv.org/abs/2606.06479

**12. Regret Minimization with Adaptive Opponents in Repeated Games**  
Authors: Mingyang Liu, Asuman Ozdaglar, Tiancheng Yu, Kaiqing Zhang  
Standard external regret in online learning fails to capture how opponents adapt based on play history. This paper introduces Repeated Policy Regret (RP-Regret), a game-theoretic metric measuring the difference between realized and best-in-hindsight accumulated utility when all players could have adapted. The key insight is that an adaptive opponent who can respond to your historical play requires fundamentally different learning guarantees than i.i.d. opponent models. Algorithms achieving sublinear RP-regret are developed and shown to be necessary for strategic interaction in real-world multi-agent settings like auctions and repeated security games.  
`cs.LG, cs.AI, cs.GT` | https://arxiv.org/abs/2606.06486

**13. Event Detection for Parameter-to-KPI Dependency Learning for AI-RAN**  
Authors: Christie Djidjev, Nicholas Kaminski  
Next-generation wireless networks rely on multiple concurrent AI-driven control functions that optimize different network objectives simultaneously, and when these functions interact they can interfere in ways difficult to detect from raw data alone. This paper presents an event detection framework for learning parameter-to-KPI dependencies in AI-RAN architectures, identifying how specific configuration changes cascade through the system to affect key performance indicators. The approach enables proactive identification of harmful AI function interactions before they cause service degradation, critical for deploying AI-RAN in production carrier networks.  
`cs.LG` | https://arxiv.org/abs/2606.06459

### 🔒 Security & Cybersecurity

**14. Will the Agent Recuse Itself? Measuring LLM-Agent Compliance with In-Band Access-Deny Signals**  
Authors: Thamilvendhan Munirathinam  
As autonomous LLM agents hold real credentials and operate infrastructure without human oversight, operators have no standard way to signal that a resource is off-limits. This paper proposes the Recuse Signal — a lightweight in-band deny signal that servers emit over existing protocol channels (SSH banners, PostgreSQL connections) to tell agents to recuse themselves without hard-failing. The study measures compliance across major LLM agents and finds significant variation in adherence, with implications for credential hygiene and infrastructure access control in agentic deployments.  
`cs.CR, cs.AI` | https://arxiv.org/abs/2606.06460

**15. WebMCP Tool Surface Poisoning: Runtime Manipulation Attacks on LLM Agents**  
Authors: Lin-Fa Lee, Yi-Yu Chang, Chia-Mu Yu, Kuo-Hui Yeh  
WebMCP enables websites to expose tools directly to AI agents, bypassing traditional user interfaces and introducing new attack surface. This study identifies Mid-Session Tool Injection (MSTI), where attackers leverage third-party scripts to inject malicious tools during an active agent session. Since WebMCP tool surfaces are dynamically exposed during sessions, a compromised third-party script can add malicious tools after the session has begun — invisible to both user and agent. The paper demonstrates practical attack feasibility and discusses mitigation strategies for the emerging WebMCP ecosystem.  
`cs.CR` | https://arxiv.org/abs/2606.06387

**16. Credential Disclosure in (EU) Digital Identity Wallets: Privacy Risks and Practical Mitigations**

The European Union will introduce the EUDI Wallet by late 2026, allowing users to hold digital identity credentials on their devices and disclose identity attributes to websites. This paper conducts a large-scale survey on credential disclosure decisions with users and experts, identifying risks from poor disclosure choices that leak sensitive identity information. It maps the privacy attack surface of the EUDIWallet architecture and proposes practical mitigations including attribute minimization, selective disclosure, and usage logging — directly informing the wallet's privacy-preserving design before deployment.  
`cs.CR` | https://arxiv.org/abs/2606.06354

Authors: Parsa Memarzadehsaghezi, Zahra Hashemi, Pooria Madani, Mehran Ebrahimi  
Evasion attacks present a significant challenge to ML-based classifiers in fraud detection and cybersecurity, but existing defenses suffer from limited generalizability across diverse attack scenarios. RESA introduces a robust ensemble framework that selectively strengthens and augments predictors against evasion attacks, systematically improving model robustness across diverse attack patterns. The approach identifies the most vulnerable ensemble members under attack and applies targeted augmentation and strengthening to those components, maintaining accuracy on clean data while dramatically improving adversarial robustness.  
`cs.CR` | https://arxiv.org/abs/2606.06265

**18. SecRL-Prune: Structured Reinforcement Learning-Based Pruning of CodeLLMs for Preserving Adversarial Code Mutation**

Large CodeLLMs can generate functionality-preserving code mutations that evade signature-based malware detection — but it was unclear whether model compression preserves this mutation capability. SecRL-Prune is a structured pruning framework for CodeLLMs that operates on feed-forward channels, applying RL-based channel selection to preserve adversarial mutation capability under compression. This enables deploying compressed CodeLLMs in security analysis pipelines with limited hardware budgets while maintaining the mutation diversity that makes them useful for adversarial code generation tasks.

**19. Steering LLM Viewpoints through Fabricated Evidence Injection**

As chatbots increasingly influence daily decision-making, their potential to produce misleading responses poses substantial risks. Ghostwriter is a two-phase attack that first repackages misleading statements with fabricated rationales, then instructs target LLMs to incorporate them into responses — exploiting the cognitive vulnerability of LLMs to uncritically trust external context bearing markers of credibility. The attack achieves high success rates in steering model viewpoints on contested topics without triggering refusal mechanisms, demonstrating a critical gap between current alignment and production deployment reality.

Authors: Qi Hu, Yifeng Tang, Qinghua Wang, Lanyang Zhao, Pengji Zhang  
Large language models are increasingly deployed as coding agents, shifting safety from individual responses to action sequences. Financial AI agents often fail because they make users carry complexity — repeatedly restating goals, risk preferences, portfolio context, past judgments, and shifting market assumptions.
