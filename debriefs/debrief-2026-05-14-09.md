## ⚡ Quick Summary

- **PersonalAI 2.0**: Knowledge graph traversal + planning mechanism boosts LLM agents, achieving 89% info-retention on MINE-1 benchmark and 4% average gain across 6 benchmarks, significantly reducing hallucination rates.
- **FrameSkip**: Vision-language-action frame selection via action variation scoring improves VLA training success rate from 66.5% to 76.15% while retaining only 20% of unique frames, streamlining robot data efficiency.
- **SafeHarbor**: Hierarchical memory-augmented guardrails for LLM agents achieve 93%+ refusal against malicious requests while maintaining 63.6% benign utility — solving the over-refusal tradeoff in agent safety.
- **Orthrus**: Dual-view diffusion/autoregressive LLM architecture delivers 7.8x inference speedup with lossless accuracy by sharing KV cache between AR and diffusion decoding heads.
- **ActGuide-RL**: Agentic RL without heavy SFT: injecting action guidance as scalable plan-style references improves GAIA by +10.7pp and XBench by +19pp using readily available action data.
- **MemReread**: Memory-guided rereading with RL-based length extrapolation enables linear-complexity long-context reasoning, recovering indirectly discarded facts via non-linear document traversal.
- **MinT**: Managed LoRA infrastructure supporting million-scale adapter catalogs at cluster scale, validated beyond 1T parameters with 18.3x step reduction via adapter-only model handoff.
- **HAGE**: RL-driven weighted graph evolution for agentic memory: query-conditioned edge routing outperforms static retrieval, improving long-horizon reasoning accuracy and efficiency.
