⚡ Quick Summary

  • Persistent Visual Memory (PVM) for LVLMs: a new lightweight module that counteracts 'Visual Signal Dilution' in large vision-language models, where visual attention decays as text history grows, enabling sustained multimodal perception.
  • Security case study revealing RAG chatbots in patient-facing medical AI can inadvertently expose backend architecture: a wake-up call for governance and data hygiene in healthcare AI deployments.
  • Position paper arguing agentic AI orchestration layers should be Bayes-consistent: treating tool selection, expert consultation, and resource allocation as explicit Bayesian decisions under uncertainty.
  • New framework for assessing when LLM tool-calling actually helps vs. hurts: characterising the tradeoff between external information benefit and internal knowledge integration noise, especially for web search.
